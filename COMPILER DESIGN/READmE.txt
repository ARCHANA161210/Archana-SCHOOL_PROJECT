This project is the whole compiler for the own design language simpl

Two folders
1.c++ code: This contain all source file and header file for own compiler.
2.simpl: This contain download version of the compiler file to run the code.
steps to run the program
1.program file should save with .sim extension.
2.download the compiler from the above link
3.Go the directory from the command prompt and follow these steps.
4.Save the directory upto simpl/bin
5.\simpl name of the file.sim.
6.\jasmin name of the file.j.
7.\run name of the class of the program.

note: install java virtual machine in the system.
