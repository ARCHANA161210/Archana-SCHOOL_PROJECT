Simpl for Windows, and Linux

Simpl Installation

To install the Simpl software, extract the Simpl directory from the archive, onto your hard-disk, at the desired location. On Linux, you may also need to change the file permissions for the scripts in the “bin” directory, to include execute permission.

Distribution Contents

The Simpl folder contains the following sub directories.

  • Bin – scripts for running the Simpl software.

  • Doc – documentation on the Simpl language.

  • Examples – a few example Simpl programs.

  • SymTab – C++ code for a symbol table class.

  • Lib – classes, forming the Simpl software.

  • Tests – a benchmark suite, used to test out the Simpl compiler.

Using the Simpl Software

To run the Simpl software, you need to have the Java JRE installed on your machine.

Making Life Easier

The commands to execute the Simpl software will not be recognized by your operating system, unless, either, you put the Simpl “bin” directory on your path variable, or you use the full path for the command, when running it. We will explain a shortcut for the second option.

You can define an environment variable, which points to the path to the Simpl commands. This saves typing when running the commands. Here is the procedure.

  1. Start up a shell. In Linux you would start up a terminal, and in Windows you would start up a command tool.

  2. Navigate to the directory where you installed the Simpl software, and then to the “bin” directory. Let us assume that you have installed Simpl to the directory c:\simpl, on Windows, or ~/simpl on Linux. To navigate to the “bin” directory in windows, you would execute the command
    cd c:\simpl\bin

    In Linux, it would be
    cd ~/simpl/bin

  3. We now set a variable, “sd”, to point to this directory. In Windows,
    set sd=%cd%

    In Linux,
    set sd=`pwd`

Running the Simpl Compiler

To compile a Simpl program, create a file, with the extension “.sim”. For the purposes of this discussion, we will assume that the file is c:\prog.sim, for windows, and ~/prog.sim for Linux.

  1. Move to the directory where the program is located.
    cd c:/
    or
    cd ~

  2. Run the compiler.
    %sd$\simpl prog
    or
    $sd/simpl prog

This will create a file c:\prog.j, or ~/prog.j.

Assembling the Program

You now assemble the program.

%sd%\jasmin prog.j

or

$sd/jasmin prog.j

This produces a flie c:\prog.class, or ~/prog.class.

Running the Program

You can now run the program.

%sd%\run prog

or

$sd/run prog

This will pop up a window that allows you to perform I/O with your Simpl program.
