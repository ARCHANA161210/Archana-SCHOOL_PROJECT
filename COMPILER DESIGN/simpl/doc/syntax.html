SIMPL Syntactic Analysis

James Gil de Lamadrid



The following is the LL(1) grammar for the SIMPL language. (Note: the \ indicates a meta-character is being used as a literal, and the # indicates locations in which syntax tree nodes are constructed.

<prog> ::= <id> \( <varlist> \) \| <varlist> \| <statlist> ! <eof> #
<varlist> ::= <id> <varlist> # | l #
<statlist> ::= <stat> <statlist> # | l #
<stat> ::= <assign> | <ifstat> | <whilestat>
<assign> ::= <id> <- <expr> ! #
<ifstat> ::= if <expr> ? <statlist> : <statlist> ! #
<whilestat> ::= while <expr> ? <statlist> ! #
<expr> ::= <aexp> <exprtail>
<exprtail ::= = <aexp> # <exprtail> | > <aexp> # <exprtail> | l
<aexp> ::= <mexp> <aexptail>
<aexptail> ::= + <mexp> # <aexptail> | - <mexp> # <aexptail> | l
<mexp> ::= <atom> <mexptail>
<mexptail> ::= * <atom> # <mexptail> | / <atom> # <mexptail> | l
<atom> ::= <id> # | <num> # | \( <expr> \)
