The following table gives the token types in the SIMPL language. (Note: a \ indicates that a meta-character is being used as a literal.)

id letter (letter | digit | _)*
num digit+
letter a | b | c ... | z | A | B | C ... | Z
digit 0 | 1 | 2 ... | 9
