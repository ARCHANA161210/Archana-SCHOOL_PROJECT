Description

SIMPL is a language that manipulates counting numbers. A SIMPL program has the form:

progName(inputVariables) |localVariables | body !
Both the input variable list and the local variable list are lists of variable names, separated by spaces. The body of the program is a list of statements. There are three statement types: assignment, conditional, and iterative. The assignment statement has the form:
variableName <- expression !
The conditional has the form
if expression ? thenStatements : elseStatements !
The iterative statement has the form:
while expression ? loopStatements !
Expressions are composed of the relational operators > and =, the arithmetic operators +, -, *, and /, with the usual meanings and order of evaluation, variable names, and counting number constants. The - operator in SIMPL represents monus, rather than minus.

A SIMPL program is executed by reading in values for the input variables, executing the body of the program, and outputting the return value. The return value of the program is the last value assigned to the program name.

Installation

The SIMPL compiler compiles SIMPL code to the jasmin assembly language. The jasmin target code must then be assembled using the jasmin assembler. The output of the jasmin assembler is a java .class file, which can then be run on the java virtual machine, supported by the java class library and the SIMPL class library.

The installation of SIMPL requires the correct installation of both java, and jasmin. Once these are correctly installed, install the simpl directory on the hard drive. Then modify the PATH environment variable, to include the appropriate simpl directories. As an example, on a PC system you might set them as follows:

set PATH= %PATH%;c:\simpl\bin
On UNIX you might set them as follows:
setenv PATH ${PATH}:/usr/local/simpl/bin
Use

After setting the environment, two batch files can be used to run SIMPL. To compile a SIMPL program file, called prog.sim, change the directory to the parent directory of prog.sim, and type the command

simpl prog
This will create a jasmin file called prog.j, which can be assembled with the command
jasmin prog.j
This will create a .class file with the same name as the SIMPL program name. Let us assume that the program name is the same as the file name. Then, to run the SIMPL program, type
run prog
This causes a GUI interface to appear. The user is prompted for input, the output is displayed, and the GUI session end when the user clicks the close box of the GUI window.

Depending on the location of the simpl directory the CLASSPATH settings in the batch files run.bat and simpl.bat may require modification, to run properly.
